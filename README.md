# Artificial Intelligence & Machine Learning (AIML) Lab

Welcome to the **Artificial Intelligence & Machine Learning (AIML)** repository. This repository contains implementations of various Machine Learning models, Data Preprocessing workflows, and Image Processing techniques.

## 📌 Repository Overview

The repository covers multiple core domains of AI and ML, implemented via Python scripts and tested with standard datasets:
1. **Machine Learning Algorithms**: Linear Regression, Logistic Regression, Support Vector Machines (SVM), and Decision Trees.
2. **Image Processing**: Morphological operations (dilation, erosion, opening, closing), grayscale conversion, and histograms.
3. **Data Analysis & Cleaning**: Data preprocessing, feature analysis, and operations on CSV datasets.

---

## 📂 Key Files & Dataset Details

### 1. Machine Learning & Analysis Scripts
- `ass2.py`: Assignment script covering machine learning model testing.
- `5images_grayscale.py`: Basic image loading, processing, and grayscale conversion.

### 2. Image Assets & Outputs
- `closing.jpg` / `closing_color.jpg`: Demonstration of morphological closing operations.

### 3. Datasets (CSV)
- `Bollywood Box Office Collection (2019-2023).csv` & `bollywood2023.csv`: Box office dataset for regression analysis.
- `books.csv`: Dataset utilized for recommender systems or metadata analysis.
- `clean_data.csv`: A preprocessed, clean subset of training data.

---

## 🛠️ Prerequisites

To run these scripts locally, install the required Python libraries:

```bash
pip install numpy pandas matplotlib scikit-learn opencv-python
```

---

## 🚀 How to Run

Clone the repository and run any of the Python scripts:

```bash
# To run grayscale processing
python 5images_grayscale.py

# To run data analysis/ML scripts
python ass2.py
```
